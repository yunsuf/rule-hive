## Quick Start

### Direct Workflow

```bash
# Start interactive session
workflow brainstorming

# With context document
workflow design-thinking --data /path/to/context.md
```

### Agent-Facilitated

```bash
# Load agent
agent cis/brainstorming-coach

# Start workflow
> *brainstorm
```
